Project Description:
437 Air is a web-based in-flight infotainment system designed to improve the passenger
experience during a flight. The application allows passengers to browse entertainment content,
create an optional user profile, customize display preferences, order food, shop for in-flight
products, and access passenger services through a simple and user-friendly interface.

Main Features:
- Guest access without requiring login
- Optional passenger profile creation
- User preferences for light mode and text size
- Entertainment catalog with movies, TV shows, and music
- Trending and recommended entertainment categories
- Favorites system for saving entertainment and other content
- Home dashboard with quick navigation
- In-flight food and beverage ordering
- In-flight shopping catalog
- Passenger services page

Running the Project:

1. Open the project folder.
2. Install the required dependencies:

   npm install

3. Start the development server:

   npm run dev

4. Open the local URL displayed in the terminal http://localhost:5173
